# 🍩 TDL (The Last Donete) - O Último Combate Açucarado

**Donut Defenders** é um jogo estilo *Space Defenders* desenvolvido em Python, onde os últimos dois donuts do mundo enfrentam hordas de bocas famintas que querem devorá-los. Prepare-se para defender sua deliciosa existência!

## 🎮 Sobre o Jogo

Você controla dois donuts heroicos tentando sobreviver no espaço, enquanto ondas de bocas inimigas tentam comê-los. Use reflexos rápidos, estratégias de movimento e ataques açucarados para defender os donuts até o fim!

## 🕹️ Como Jogar

- **Movimentação**: Use as teclas "WASD" e as setas para a o segundo jogador para mover os "donetes".
- **Ataque**: Use a barra de espaço e "leftshift" para o segundo jogador para disparar confeitos contra as bocas.
- **Objetivo**: Sobreviver o máximo de tempo possível e derrotar o maior número de bocas.

## 💻 Requisitos

- Python 3.8 ou superior
- Biblioteca `pygame` instalada
  - Instale com: `pip install pygame`

## ▶️ Como Executar

1. Certifique-se de ter o Python instalado.
2. Instale a biblioteca `pygame`.
3. Execute o arquivo principal:

```bash
python TDL.py
